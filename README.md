# 📘 README.md
##🛠️ Dotfiles

Personal configuration files for my Linux setup.  
This repository contains my customized environment configs to improve productivity, aesthetics, and workflow.

## 📂 Included Configurations

- **alacritty** → Terminal emulator configuration  
- **btop** → System resource monitor setup  
- **cava** → Audio visualizer config  
- **i3** → Window manager configuration  
- **neofetch** → System info display customization  
- **picom** → Compositor settings (transparency, animations, etc.)

## 🚀 Setup

Clone the repository:

```bash
git clone https://github.com/your-username/dot-files.git
cd dot-files
```
